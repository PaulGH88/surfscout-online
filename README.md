# surfScout.online

Interactive **Wave Radar** for the Market Surfing doctrine.

A single-page site where a visitor scores a problem-wave on 10 dimensions and gets
back a Catch / Scout / Park verdict plus a personalized Wave Charter. The doctrine
is the copy wrapped around the instrument.

- **Catch** 35+ - enter with a narrow paid wedge. Keep the board, not the wave.
- **Scout** 25-34 - watch it. Hunt the forcing function.
- **Park** under 25 - ride as joy if you must, not as strategy.

> Let the wave go. Keep the board. Burn the anchors.

## Build
Single self-contained `index.html` - inline CSS + vanilla JS, no backend, no build
step. All scoring and Wave Charter generation run client-side.
